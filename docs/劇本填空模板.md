## 2) 快速條件對照（中文）

- `require_party_non_empty`：要有出戰寵物才顯示該選項  
- `require_party_empty`：沒有出戰寵物才顯示  
- `require_in_homestead`：人在家園才顯示  
- `min_affinity=50`：好感至少 50  
- `require_grant_once_pending=abc`：獎勵 `abc` 還沒拿過才顯示  
- `require_grant_once_done=abc`：獎勵 `abc` 已拿過才顯示  
- `target_node_id=__CLOSE__`：選了就關閉對話

## 3) 最小可跑範例（可照抄）

- 節點代號：`greet`
- 台詞：`今天想做什麼？`
- 選項：
  1. `給我一顆石頭` -> `need_help`（條件：`require_grant_once_pending=lakeside_stone_intro`）
  2. `最近如何？` -> `chat`（條件：`require_grant_once_done=lakeside_stone_intro`）
  3. `先這樣` -> `__CLOSE__`

- 節點代號：`need_help`
- 台詞：`拿去吧，小心砸到腳。`
- 進入效果：`GIVE_ITEM:mat_stone_01:1`，`ADD_AFFINITY:+10`，`grant_once=lakeside_stone_intro`
- 選項：
  1. `謝謝` -> `__CLOSE__`（主角 happy：是；頭上提示：`獲得了道具: "小石頭" !`）
