# 家園翻土土格暫時移除與還原

## 目前狀態（暫時）

為方便在**嘟嘟動畫未完成**時仍可測採收，場景內**已移除** `HomesteadSoilPlot`（需寵物翻土 → 計時 → 成熟），改為 **8 株預設成熟的 `HomesteadCrop`**（與 Phase 10 第一階相同行為：進採收模式後滑掃入包）。

已修改的場景：

| 場景 | 路徑 |
|------|------|
| 湖畔關卡（主流程家園區） | `scenes場景/levels/lake_side/LakeSideLevel.tscn` → `HomesteadBundle/Crops` 下 `MatureCrop01`～`08` |
| 獨立家園關卡（換關用） | `scenes場景/levels/homestead/HomesteadLevel.tscn` → `Crops` 下 `MatureCrop01`～`08` |

**未刪除的程式／資源**（嘟嘟做好後仍可直接用）：

- `scenes場景/levels/homestead/HomesteadSoilPlot.tscn`
- `src腳本/entities/homestead/HomesteadSoilPlot.gd`
- `GlobalBalance.HOMESTEAD_CROP_GROW_SEC`、`PetCompanion` 翻土邏輯、`dudu_pet.tres` 等

---

## 嘟嘟動畫做好後：建議還原步驟

1. **開啟** `LakeSideLevel.tscn`（與若有用到的 `HomesteadLevel.tscn`）。
2. 在 **`HomesteadBundle/Crops`**（或 `HomesteadLevel` 的 **`Crops`**）下：
   - **刪除**（或停用）`MatureCrop01`～`MatureCrop08`。
   - **實例化** `HomesteadSoilPlot.tscn`，數量與位置依美術編排（先前曾為 3 格；可增減）。
3. 在檔案系統將 **`HomesteadSoilPlot.tscn`** 拖進場景樹時，Godot 會自動加 `ExtResource`；若手改 `.tscn`，請對照 Git 歷史或下列引用：
   - `ext_resource`：`res://scenes場景/levels/homestead/HomesteadSoilPlot.tscn`（`PackedScene`）
4. **出戰嘟嘟**（`dudu_pet.tres`，已含翻土技能）驗收：翻土 → 約 1.5 秒 → 掃採。
5. 若希望**同時保留**「立刻可採的展示株」與「翻土田」，可只還原部分土格，並**保留少數** `HomesteadCrop`（`is_mature = true`）作裝飾或教學。
